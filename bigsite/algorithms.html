Algorithms

An algorithm is a finite sequence of well defined instructions used to solve a problem or perform a computation. Classic examples include sorting algorithms such as quicksort and mergesort, and searching algorithms such as binary search.

The efficiency of an algorithm is described using big O notation, which measures how the running time grows as the input size increases. Recursion, where a function calls itself on smaller inputs, is a powerful technique behind many algorithms.

Data structures like hash tables, trees, and graphs are chosen to make specific algorithms fast and memory efficient.
