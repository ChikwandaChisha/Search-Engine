Python

Python is a high level programming language known for its readable syntax and significant indentation. It supports multiple paradigms including procedural, object oriented, and functional programming.

Python is widely used for web development, data science, automation, and teaching because its code is concise and easy to read. A large standard library and thousands of third party packages make it flexible for many algorithms and applications.

The simple http server that serves this website is itself written in Python.
